# Community Center Website

A responsive and user-friendly **frontend website** developed for a Community Center.  
This project focuses on providing clear information, easy navigation, and a modern design to help visitors learn about activities, services, and events of the community center.

##Features
- Responsive design for desktop, tablet, and mobile devices  
- Clean and modern UI  
- Interactive elements using JavaScript  
- Organized layout for community information  
- Easy navigation and accessibility-friendly structure  

## Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior
